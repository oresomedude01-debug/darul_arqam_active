feat: Professional Blog Performance, Auth, and Mobile Navigation Improvements

## Features Added

### 1. Smart Blog Caching with Intelligent Invalidation
- New `BlogCacheService` for centralized cache management
- Automatic cache invalidation on blog create/update/delete
- Category-based caching with TTL of 1 hour
- Service worker stale-while-revalidate strategy for blog content
- Fresh blog content guaranteed after updates

### 2. Session Expiry Handling
- New `HandleSessionExpiry` middleware for automatic logout
- Auto-logout after 120 minutes of inactivity
- Session regeneration for enhanced security
- User-friendly "session expired" redirect with message
- Configurable timeout via SESSION_LIFETIME env var

### 3. PWA Persistent Authentication
- New `PWAAuthService` with two-token system
- Encrypted access tokens (30-day lifetime)
- Refresh tokens (60-day lifetime) for automatic renewal
- HttpOnly, Secure, SameSite cookies for XSS/CSRF protection
- Automatic token refresh before expiry

### 4. PWA Authentication API Endpoints
- POST `/api/pwa/auth/login` - Generate persistent tokens
- GET `/api/pwa/auth/me` - Get authenticated user info
- POST `/api/pwa/auth/refresh` - Refresh access token
- POST `/api/pwa/auth/logout` - Revoke all tokens
- POST `/api/pwa/cache/invalidate` - Admin cache control

### 5. Mobile-First Public Navigation
- Complete redesign of public navigation layout
- Responsive hamburger menu for mobile
- Slide-out navigation drawer with smooth animations
- Touch-optimized tap targets (48px minimum)
- Responsive grid: 1 column (mobile) → 3 columns (desktop)
- Session expiry alert display

### 6. Modern Blog Page UI/UX
- Card-based responsive blog layout
- Category filtering with sticky controls on mobile
- Featured images with gradient overlays
- Color-coded category badges
- Reading time and author metadata
- Smooth hover effects and animations
- Empty state messaging
- Newsletter subscription CTA

### 7. Service Worker Caching Strategies
- Stale-while-revalidate for blog content (fresher than cache-first)
- Network-first for API and dynamic content
- Cache-first for static assets and pages
- Automatic cache cleanup on activation
- Smart cache versioning for production deployments
